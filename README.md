## 🚧 Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/BrianPutallazExequiel/brianexequiel.git
   ```
2. Install dependencies:
    ```bash
    pnpm install
    ```
3. Run in development mode:
    ```bash
    pnpm run dev
    ```
